## 项目功能
1. 顾客端：登录/注销、商品浏览、购物车管理、订单提交/查看；
2. 商家端：商品增删改、订单管理。

## 项目结构
CSS/：存放样式文件（全局样式、页面单独样式）；
js/：存放交互逻辑文件（初始化、购物车、结算等）；
HTML文件：首页、登录页、顾客中心、商家中心等页面入口。
